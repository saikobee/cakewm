* Display[screens]
  - Move window to nth screen
  - Swap current tag with nth screen

  * Screen[tags]
    - Move window to nth tag
    - Select nth tag

    * Tag[columns]
      - Change (cycle) the amount of columns
      - Move window to nth column
      - Select nth column

      * Column[stacks]
        - Move window to nth stack
        - Select nth stack

        * Stack[windows]
          - Select next window
          - Select prev window
          - Move window to next position
          - Move window to prev position
          - Add window
          - Close window

          * Window
